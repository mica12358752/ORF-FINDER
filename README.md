# ORF-Finder

A Python-based tool designed to identify Open Reading Frames (ORFs) in DNA sequences. This project was developed as part of a bioinformatics workflow to automate the search for potential protein-coding regions.

 Features
* **Sequence Analysis:** Scans DNA sequences for start and stop codons.
* **Flexible Parameters:** Customizable minimum length for ORF detection.
* **Batch Processing:** Ability to handle multiple sequences (if applicable).
